# متجر كتب إلكترونية بسيط (E-books Store)

هذا مشروع تجريبي بسيط لبيع الكتب الإلكترونية. يتكون من واجهة ثابتة وسيرفر Node.js الذي يتعامل مع Stripe لإنشاء جلسة دفع ثم يمنح رابط تنزيل مؤقت بعد الدفع.

متطلبات:
- Node.js 16+
- حساب Stripe (مفتاح سري ومفتاح منشور)
- ملف كتاب (مثلاً PDF) في مجلد `downloads/`

إعداد سريع (تشغيل محلي):

1. انسخ المشروع، ثم ثبّت الحزم:
   npm install

2. إعداد المتغيرات البيئية:
   - STRIPE_SECRET_KEY : مفت
